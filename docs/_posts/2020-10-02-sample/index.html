Consistency#

Sample error depends on sample size:

  • The sample approaches the population as the the sample size increases

  • Attributes concentrate more around the population value as the the sample size increases

The consistency for an attribute is shown by the concentration around the true population value in the sample. We use the absolute difference between the sample attribute and the population attribute to quantify this concentration, $$\lvert a(\mathcal S) -a(\mathcal P) \rvert = \lvert \frac{1}{n}\sum_{y\in\mathcal{S}}y_u - \frac{1}{N}\sum_{y\in\mathcal{P}}y_u \rvert < c$$ for câ€„>â€„0
